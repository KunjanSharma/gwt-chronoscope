Elements

Element Description Child Elements Properties Sample
series
for each time series, there is a series element with class s#, for example, the first time series dataset is represented by series.s0, and the second by series.s1
    • visibility
    • background-image
    • background-color
    • group
    • color
    /* Make the second times series line green */
series.s1 line {
 color: green;
}
    marker
    A marker that represents a single point on the curve.
    • domainmarker
    • visibility
    • opacity
    • background-color

marker {
 background-color: green;
 opacity: 0.3;
}

          grid
          In a line plot, allows control of fill region beneath line.
            • background-image
            • opacity
            • background-color

 background-color: lightblue;
}

              Properties

              Property Name Description Units
              value-visibility
              		CSS value-visibility property controls whether value lables are drawn or notSTRING
              tick-position
              		A value of 'inside' puts tick labels inside the plot, a value of 'outside' renders them outside the plotSTRING
              visibility
              		CSS visibility property controls whether a chart component is drawn or notSTRING
              tick-align
              		A value of 'above' renders tick labels at a position numerically greater than the tick. A value of 'middle' centers the label on the tick. A valuer of 'below' renders tick labels at a position numerically lesser than the tick's value.STRING
              icon-width
              		In the legend axis panel,set the icon width of the legend labelsLike this: icon-width: 12pt or icon-width:auto STRING
              background-color
              		Standard CSS background-color propertyCOLOR
              point-selection
              		Specifies 'nearest' or 'domain' for hover selection, nearest selects the closest data point based on the (x,y) of the mouse, 'domain' selects the data points tothe X position of the mouse.STRING
              display
              		override display type for series, e.g. line, bar, stepSTRING
              font-weight
              		Standard CSS Font Weight propertySTRING
              icon-height
              		In the legend axis panel,set the icon height of the legend labelsLike this: icon-height: 12pt or icon-height:auto STRING
              column-count
              		In the legend axis panel,set the column count of the legend labels in one lineLike this: column-count: 3 or column-count:auto STRING
              line-thickness
              		Specifies the thickness of lines drawn on the chartPX
              color
              		A standard CSS color propertyCOLOR
              font-size
              		Standard CSS Font Size propertyPT
              column-width
              		In the legend axis panel,set the column width of the legend labelsLike this: column-width: 12pt or column-width:auto STRING
              date-format
              		Specifies the date format for labels which contain datesSTRING
              opacity
              		Specifies the opacity value from 0 (transparent) to 1 (opaque)FLOAT
              radius
              		For circular points, specifies the radius of circlePX
              font-family
              		Standard CSS Font Family propertySTRING
              group
              		Asserts that a series is to be grouped with other series having the same group id into a new composite dataseries.STRING
              number-format
              		Specifies the number format for values displayedSTRING
              background-image
              		CSS Background image property with extensions for gradientsBGIMAGE

              Units

              Unit TypeDescription
              INTEGER
              		An integer value.
              PX
              		An integer value in pixel units, with optional px suffix
              FLOAT
              		A floating point value
              PT
              		A font point size, a number followed by the suffix 'pt'
              URI
              		A URL of the format url(http://host.domain/some/path)
              COLOR
              		A CSS color name, rgb triple rgb(r,g,b), hexadecimal triple (e.g. #f0a2c9), an rgba quadruple rgba(r,g,b,a). Note: Shorthand colors such as #fff; will work in most browsers but fail in IE.
              BGIMAGE
              		A url to an image to be placed in the background of the form url(http://host.domain/some/image.png) or a gradient function gradient(startx, starty, endx, endy, colorstop, color, colorstop, color, ...)
              STRING
              		A string with or without surrounding quotes.
